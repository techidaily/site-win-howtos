Destiny 2 Launch Issue Resolved: Conquering the Persistent Initializing Error

When you pay the money and look forward to enjoying the game but find that the game is stuck on initializing. You must be annoyed. Don’t worry, you’re not alone. There are ways to solve this problem.

Before you start, check your connections first. If this problem is not caused by internet connections, you can try the below methods to solve this issue.

Try these methods

  1. Rename the Destiny 2 folder
  2. Reinstall the Battle.net app
  3. Change Blizzard Settings
  4. Run your Blizzard client as administrator
  5. Check your antivirus software
  6. Set the Windows Secondary Logon is set to automatic

Method 1: Rename the Destiny 2 folder

To solve Destiny 2 stuck on initializing problem, this method is the first to go. Many users have proven its effective.

  1. Pause the initializing process and closeBattle.net .
  2. Navigate to your Destiny 2 folder location. Usually, it’s in C:\Program Files (x86)\. Then rename the Destiny 2 folder.
  3. OpenBattle.net and clickInstall .
  4. Change the folder name back to Destiny 2.
  5. ClickStart install .
  6. Be patient, the process will resume from where you left off after verifying the files that are already there. Then you’re good to go.

After downloading the program successfully, it is recommended toupdate your driver to improve your gaming performances.

Method 2: Reinstall the Battle.net app

Reinstall the Battle.net can may help you solve the issue.

  1. Uninstall the Battle.net app.
  2. Reinstall Battle.net app and load the game. The update will stay on ‘Initialising’ for maybe 3-5min and will then start ‘Finalizing’.
  3. After finalizing, you should be able to play the game.

Method 3: Change Blizzard Settings

Change the Blizzard’s download settings may help you increase the download speed.

  1. Click on the Blizzard logo on the upper right side. Then selectSettings .
  2. Select theGame Installs/Update . Scroll down to find the Network Bandwidth.
  3. Change the Max Download rate by setting the Latest Updates and Future Patch updates to 0 to get the max download rate. Then PressDone to apply the settings.
  4. Download Destiny 2 to see whether it can help.

Method 4: Run your Blizzard client as administrator

The privilege issue might be the reason for this problem. With high integrity access, Blizzard can make full use of its features, so run Blizzard as an administrator to see if this fixes your issue.

  1. Exit Blizzard.
  2. Right-click on the Blizzard icon and clickProperties .
  3. Under the Compatibility tab, tick Run this program as an administrator . Then click OK .
  4. Run Blizzard and check whether the problem is solved or not. SwifDoo PDF 2-Year Plan

Method 5: Check your antivirus software

Sometimes your antivirus software may be the reason for the Destiny 2 stuck on initializing. So disable your antivirus software and try downloading again to check if the issue persists.

If this resolves your problem, you may need to replace your antivirus software or contact the vendor of the software for advice.

IMPORTANT : Be extra careful about what sites you visit, what emails you open and what files you download when your antivirus software is disabled.

Method 6: Set the Windows Secondary Logon is set to automatic

Windows Secondary Logon allows administrators to log on with a non-administrator account to perform administrative tasks without logging off. The Secondary Logon service must be enabled for Blizzard games to install and run.

  1. Press theWindows logo key +R to open the Run box.
  2. Type “services.msc” and pressEnter .
  3. Locate and right-clickSecondary Logon in the right pane.
  4. SelectProperties .
  5. Find Startup type and selectAutomatic in the dropdown menu. Then clickOK .

Some security programs may turn the secondary logon service off. If the service is disabled again after you’ve switched it to automatic, you must configure your security program to stop disabling the service. Contact your security program’s manufacturer if you need assistance with configuring your security software.
