**Solution 1: Restart Driver Easy**

 The problem can be caused by temporary disconnection from the server. When you run into this error, you can try to restart Driver Easy and see if the problem resolves.

 **Solution 2: Ping Checking**

 1\. Press **Win+R**   (Windows key and R key) combination to open the Run dialog box.  
 2\. Type **cmd**   in the run box. Click **OK**  button.

 **Solution 4: Antivirus / Firewall/VPN Checking**

 Some AntiVirus, Firewall programs and VPN may block the connection between Driver Easy and the driver database server or driver download server.

 If you are using any Antivirus or Firewall or VPN, please try to temporarily disable them and then try again.

 If the problem could not be resolved or persists, please send us with further information using Driver Easy **Feedback** tool so we could assist you to resolve the problem.
